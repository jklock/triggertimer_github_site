# Drills and Templates Guide

Create, customize, and manage your training drills.

---

## Overview

TriggerTimer includes a library of pre-built drills and lets you create custom templates for your specific training needs.

---

## Drill Categories

| Category | Description | Example Drills |
|----------|-------------|----------------|
| **Draw** | Drawing from holster | Bill Drill, First Shot Draw |
| **Reload** | Magazine changes | Emergency Reload, Speed Reload |
| **Transition** | Target transitions | Double Tap, El Presidente |
| **Accuracy** | Precision focused | Slow Fire, Dot Torture |
| **Speed** | Fast shooting | Par Time Drill, Sprint Drill |
| **Competition** | Match preparation | IDPA Classifier, USPSA Stage |
| **Custom** | User-created drills | Your templates |

---

## Browsing Drills

1. Tap the **Drills** tab
2. Use the category filter at the top
3. Scroll to find drills
4. Tap a drill for details

### Drill Details

Each drill shows:

- **Name** and **Description**
- **Category** and **Difficulty**
- **Par Time** (target time)
- **Shot Count** (expected repetitions)
- **Best Time** (your personal record)

---

## Starting a Drill

1. Select a drill from the library
2. Tap **Start Drill**
3. Follow the countdown
4. Complete the drill
5. Review your results

---

## Creating Custom Drills

### Using Drill Builder

1. Go to **Settings** → **Drills** → **Drill Builder**
2. Tap **+** to create a new template
3. Enter drill details:
   - **Name** — Descriptive title
   - **Category** — Select from list
   - **Par Time** — Target completion time
   - **Shot Count** — Number of shots expected
   - **Description** — Instructions or notes

4. Tap **Save**

### Template Settings

| Setting | Description |
|---------|-------------|
| **Par Time** | Target time to complete |
| **Shot Count** | Expected number of shots |
| **Progression Mode** | How drills advance in a session |
| **Notes** | Personal instructions or cues |

---

## Progression Modes

Control how drills advance during training sessions.

| Mode | Behavior |
|------|----------|
| **Manual** | Tap to advance to next drill |
| **Auto-Advance** | Automatically proceed after completion |
| **Random** | Drills appear in random order |
| **Timed** | Drills advance based on time |

### Setting Progression Mode

1. Open Template Editor
2. Select **Progression Mode** picker
3. Choose your preferred mode
4. Mode applies to all drills in that template

---

## Drill Sessions

### Multi-Drill Sessions

Create sessions with multiple drills:

1. Create or edit a template
2. Add multiple drills to the template
3. Set **Progression Mode**
4. Start the session
5. Complete drills in sequence

### Session Flow

1. First drill loads
2. Timer starts on your command
3. Complete the drill
4. Automatic or manual advance to next
5. Repeat until all drills complete
6. Review session results

---

## Managing Drills

### Favorites

1. Long-press any drill
2. Tap **Add to Favorites**
3. Access favorites from the Drills tab filter

### Recently Used

The Home tab shows your recently used drills for quick access.

### Deleting Custom Drills

1. Swipe left on the drill
2. Tap **Delete**
3. Confirm deletion

Note: Built-in drills cannot be deleted.

---

## Tips for Effective Training

1. **Start simple** — Master basic drills before complex ones
2. **Track progress** — Use the Data tab to see improvement
3. **Consistent practice** — Short daily sessions beat long weekly ones
4. **Progressive difficulty** — Gradually reduce par times
5. **Mix it up** — Use Random mode to challenge yourself
