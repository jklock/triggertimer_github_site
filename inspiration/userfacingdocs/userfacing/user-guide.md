## Quick Start Guide

### Your First Training Session

1. **Open TriggerTimer** on your iPhone
2. **Tap the Sessions tab** (list icon at bottom)
3. **Tap + to create a new session**
4. **Enter a session title** (or accept the auto-generated one)
5. **Add drills to your session:**
   - Tap "Add Drill"
   - Browse or search the drill library
   - Tap a drill to add it
6. **Start training:**
   - Tap on a drill in your session
   - Tap "Start" to begin timing
   - Tap "Shot" to record each trigger pull (manual mode)
   - Tap "Stop" when the rep is complete
7. **Review your results** in the session view

**Tip:** For the fastest start, use Quick Launch — long-press any template or drill and tap "Start Session Now."

## Sessions

Sessions are the core of your training history. Each session contains one or more drills, and each drill tracks individual repetitions with timing data.

### Creating a Session

**Method 1: From Scratch**

1. Go to Sessions tab
2. Tap + (plus button)
3. Enter a title or accept the default
4. Add drills from the library or create custom drills

**Method 2: From a Template**

1. Go to your templates list
2. Tap a template
3. Tap "Start Session"
4. A new session is created with all template drills pre-loaded

**Method 3: Quick Launch**

1. Long-press any template
2. Tap "Start Session Now"
3. Training begins immediately

### Inside a Session

A session displays:

- **Session title and date**
- **List of drills** with rep counts and times
- **Overall statistics** (total reps, average times)
- **Notes** you've added during training

### Recording Repetitions

For each drill in your session:

1. **Tap the drill** to expand it
2. **Tap "Start Rep"** to begin timing
3. **Record shots:**
   - **Manual mode:** Tap "Shot" for each trigger pull
   - **SG Timer mode:** Shots are recorded automatically
   - **Watch mode:** Motion detection or manual taps
4. **Tap "Stop"** to end the rep
5. **View your split times** immediately

### Adding Notes

During or after training:

1. Tap a drill or rep
2. Tap the note icon
3. Enter your observations
4. Notes are saved with your session

### Editing Sessions

- **Rename:** Tap the session title
- **Delete drills:** Swipe left on a drill
- **Reorder drills:** Long-press and drag
- **Add voice notes:** Tap the microphone icon
- **Manual time entry:** Tap any time to edit it

### Session History

All your sessions are preserved:

- Scroll through past sessions
- Filter by date range
- Search by session or drill name
- Tap any session to view full details

## Drill Library

The Drill Library is your repository of training exercises — both built-in drills and your custom creations.

### Browsing Drills

**Opening the Library:**

- From Sessions: Tap "Add Drill" when creating a session
- From anywhere: Navigate to the Drill Library view

**Navigating:**

- **Search bar** — Type to find drills by name
- **Category filter** — Tap chips for draw, reload, speed, etc.
- **Difficulty filter** — Filter by beginner to expert
- **Tag filter** — Find drills with specific tags

**Sorting Options:**

- Alphabetical (A-Z, Z-A)
- Difficulty (easy to hard, hard to easy)
- Recently used
- Favorites first

### Understanding Drill Details

Each drill shows:

- **Name** — The drill's title
- **Category** — What skill it trains (draw, reload, etc.)
- **Difficulty** — Beginner, intermediate, advanced, or expert
- **PAR Time** — Target completion time (if applicable)
- **Shot Count** — How many shots per rep
- **Default Reps** — Suggested repetitions
- **Instructions** — Step-by-step guidance
- **Tags** — Keywords for organization

### Using a Drill

From the drill detail view:

- **"Start Drill"** — Creates a quick session with just this drill
- **"Add to Session"** — Adds the drill to an existing session
- **"Add to Template"** — Saves to a reusable template
- **"Duplicate & Edit"** — Creates a copy you can customize

### Creating Custom Drills

1. Tap "Create New Drill" (+ button in library)
2. Enter drill details:
   - **Name** — What you'll call this drill
   - **Description** — Brief explanation
   - **Category** — Select from the list
   - **Difficulty** — Your assessment
   - **PAR Time** — Optional target time
   - **Shot Count** — Shots per repetition
   - **Start Delay** — Time before timer starts
   - **Instructions** — Detailed steps
   - **Tags** — Add keywords for search
3. Tap "Save"

Your custom drills appear with a different icon and can be edited or deleted at any time.

### Favoriting Drills

- Tap the heart/star icon on any drill
- Favorited drills appear at the top of your library
- Use "Favorites" filter to see only favorites

### Built-in Drill Categories

**Draw (8 drills)**

- Basic Draw, Quick Draw, Strong Hand Only, Concealment Draw, and more

**Reload (6 drills)**

- Emergency Reload, Tactical Reload, Speed Reload variations

**Transition (6 drills)**

- Multi-target engagements, horizontal and vertical transitions

**Speed (11 drills)**

- Bill Drill, failure drills, speed-focused exercises

**Accuracy (13 drills)**

- Precision work, dot drills, focus exercises

**Competition (11 drills)**

- El Presidente, classifiers, match-style drills

## Templates

Templates are reusable session blueprints. Create a template once and use it to generate consistent training sessions.

### Creating a Template

1. Go to Templates section
2. Tap + (plus button)
3. Name your template
4. Add drills:
   - Search and add from the library
   - Create custom drills inline
   - Set specific rep counts for each drill
5. Arrange drills in your preferred order
6. Save the template

### Using a Template

**Starting a Session:**

1. Tap the template
2. Tap "Start Session"
3. A new session is created with all template drills
4. Modify if needed, then begin training

**Quick Launch:**

1. Long-press the template
2. Tap "Start Session Now"
3. Training begins immediately with default settings

### Managing Templates

- **Edit:** Tap the template, then edit button
- **Duplicate:** Long-press, tap "Duplicate"
- **Delete:** Swipe left or long-press, tap "Delete"
- **Reorder drills:** Long-press and drag within template

### Template Ideas

- **Morning Fundamentals:** Basic draw + reload + one accuracy drill
- **Competition Prep:** El Presidente + Bill Drill + classifier
- **Speed Day:** All speed-focused drills
- **Weak Hand Work:** Strong hand and weak hand drill variations

### Dashboard Overview

When you open the Dashboard:

- **Widget grid** shows your selected metrics
- **Edit mode** lets you customize the layout
- **Add widgets** to track new metrics

- Average split time (last 7/30/90 days)

- Total reps this week

- Performance by drill category
- Weak area identification
- Category comparison chart

**Goal Widgets:**

- PAR achievement rate
- Streak tracker

### Customizing Your Dashboard

**Adding Widgets:**

1. Tap + (plus button)
2. Browse available widgets
3. Tap to add to dashboard

**Removing Widgets:**

1. Enter edit mode (tap Edit)
2. Tap the X on any widget
3. Confirm removal

**Rearranging Widgets:**

1. Enter edit mode
2. Drag widgets to new positions
3. Tap Done

**Configuring Widgets:**

1. Tap a widget
2. Adjust settings:
   - Time range
   - Category filter
   - Display format
3. Save changes

### Filtering Data

Apply filters to focus your analytics:

- **By category:** See only draw drills, only speed drills, etc.
- **By tags:** Filter to specific training focuses
- **By date range:** Last week, month, quarter, year, or custom
- **By drill:** Track progress on a specific exercise

## Device Connections

TriggerTimer works with external devices for enhanced training.

### Checking Device Status

Go to the **Devices tab** to see:

- **SG Timer:** Connected / Not Connected
- **Apple Watch:** Paired / Not Paired
- **Status bar:** Quick glance at connection state

### Connecting Your SG Timer

1. **Power on** your SG Timer
2. **Enable Bluetooth** on your iPhone
3. Go to **Devices tab**
4. Tap **"Scan for Devices"**
5. Your SG Timer appears in the list
6. Tap to **connect**
7. Wait for "Ready" status

Once connected:

- PAR settings push automatically when you select drills
- Shot detection begins immediately
- Split times appear in real-time

### Troubleshooting SG Timer

**Not appearing in scan:**

- Ensure the timer is powered on
- Check that Bluetooth is enabled
- Move closer to the timer
- Try restarting the timer

**Disconnecting frequently:**

- Check timer battery level
- Reduce distance between phone and timer
- Remove other Bluetooth devices that may interfere

### Apple Watch Pairing

Your Apple Watch pairs automatically if:

- Watch is paired with your iPhone in system settings
- TriggerTimer Watch app is installed
- Both devices have Bluetooth enabled

To verify:

1. Open TriggerTimer on your iPhone
2. Open TriggerTimer on your Watch
3. Check that status shows "Connected"

### Offline Mode

When no devices are connected:

- **Manual Mode** indicator appears
- All features remain available
- Use on-screen buttons for timing
- Enter shots manually
- Full functionality maintained

## Import & Export

Move data into and out of TriggerTimer.

### Importing Data

**Supported Formats:**

- PractiScore CSV exports
- TriggerTimer JSON sessions
- Generic CSV with timing data

**How to Import:**

1. Go to **Import tab**
2. Select file source:
   - Files app
   - AirDrop
   - Email attachment
3. TriggerTimer analyzes the file
4. **Preview** shows how data will be imported
5. **Map fields** if needed (for CSV files)
6. Tap **Import**

**Field Mapping:**
For CSV files, you may need to match columns:

- Which column contains times?
- Which column contains drill names?
- What's the date format?

Save mappings as templates for repeated imports.

### Exporting Data

**Export a Single Session:**

1. Open the session
2. Tap share icon
3. Choose format:
   - **CSV** — Spreadsheet-compatible
   - **JSON** — Full data backup
4. Share via AirDrop, email, Files, etc.

**Export Multiple Sessions:**

1. Go to Sessions
2. Tap "Select"
3. Choose sessions to export
4. Tap Share
5. Choose format and destination

**Export All Data:**

1. Go to Settings
2. Find "Data Management"
3. Tap "Export All Data"
4. Choose JSON (complete backup)

### Data Backup

**Manual Backup:**

1. Export all data to JSON
2. Save to iCloud Drive or other cloud storage
3. Repeat periodically

**iCloud Sync (when enabled):**

- Sessions sync across your devices automatically
- Templates and drills stay in sync
- Conflict resolution handles simultaneous edits

## Apple Watch

The Watch app provides a streamlined training interface on your wrist.

### Watch App Overview

Opening TriggerTimer on your Watch shows:

- **Active session status** (if any)
- **Quick start buttons**
- **Connection status**

### Starting Training from Watch

1. Open TriggerTimer on Watch
2. Tap a recent drill or template
3. Training begins with haptic countdown
4. Draw when you feel the final haptic
5. Tap to record shots (or use motion detection)
6. View times on your wrist

### Haptic Feedback

The Watch provides tactile signals:

- **Countdown:** 3 taps... 2 taps... 1 tap... GO
- **PAR warning:** Double tap near PAR time
- **Rep complete:** Distinctive pattern
- **Drill complete:** Series of congratulatory taps

### Motion Detection

When enabled:

- Wear Watch on your shooting hand
- The start motion of your draw triggers timing
- Natural training flow — no button needed

Calibrate sensitivity:

1. Open Settings (on iPhone)
2. Find Watch settings
3. Adjust motion detection sensitivity
4. Practice a few draws to confirm

### Heart Rate

When heart rate tracking is enabled:

- See your HR before starting a drill
- Monitor training intensity
- Review HR data in session details
- Correlate performance with physiological state

### Offline Watch Training

Train without your iPhone nearby:

1. Select drills on Watch before going offline
2. Train normally
3. Data saves on Watch
4. Syncs to iPhone when reconnected

## Troubleshooting

### Common Issues

**App won't start:**

- Force quit and relaunch
- Restart your iPhone
- Check for app updates

**Sessions not saving:**

- Ensure sufficient storage
- Force quit and reopen
- Export data and reinstall if persistent

**SG Timer won't connect:**

- Check timer is powered on
- Verify Bluetooth is enabled
- Move closer to timer
- Restart both devices

**Watch not syncing:**

- Verify both apps are open
- Check Bluetooth connection
- Force quit apps on both devices
- Re-pair Watch in system settings if needed

**Import failing:**

- Verify file format matches expected type
- Check file isn't corrupted
- Try a different export from source
- Contact support with sample file

### Getting Help

If you encounter issues not covered here:

1. Check for app updates
2. Review this guide for your specific feature
3. Try restarting the app and device
4. Contact support with:
   - Device model and iOS version
   - Steps to reproduce the issue
   - Screenshots if applicable

## Glossary

- **PAR Time:** Predetermined time limit for completing a drill
- **Split Time:** Time between individual shots
- **Repetition (Rep):** One complete execution of a drill
- **Template:** Reusable session blueprint
- **Drill Definition:** The recipe/specification for a drill
- **Session:** A training period containing drills and reps
- **Quick Launch:** Starting a session with one tap
- **Motion Detection:** Using Watch accelerometer to detect draw
